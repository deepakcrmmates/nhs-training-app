Feature Highlight

Property Search with Google Maps

  • Search by postcode + radius (1-20 miles)
  • Google Maps with numbered green pins
  • Property cards with EPC, council tax, flood risk, floor area
  • Application history badges and filter stats
  • Snapshot bar — Total Applications, This Month, This Year
  • Clickable filters on snapshot stats
  • Recent Application and Property links per card
  • Sort by distance, bedrooms, or EPC rating
  • Map ↔ card click sync

Technical Implementation

Map: Visualforce page in iframe (Locker Service workaround)
Communication: LWC ↔ VF via postMessage
Geocoding: Google Maps API with daily counter
Distance: Haversine formula (aerial)
Data: NHS_Property__c with Longitude_Latitude__s coordinates
History: Opportunity.Property__c lookup aggregation

Automation

Airparser — PDF Application Parsing

Flow

Step 1
Email with PDF sent to Airparser inbox
Step 2
Airparser AI extracts structured data
Step 3
Salesforce fetches parsed data via API
Step 4
Split-screen review: parsed data ↔ editable form
Step 5
Accept & Create Application

Smart Transformations

Name Splitting "ANTHONY WHITWELL" → First + Last
Address Parsing Street / City / Postcode auto-split
Currency "£495,000" → 495000
Housebuilder Lookup Name → Account ID
45 Default Rules Configurable mapping

Two Interfaces

Inbox (Standard Users) — simplified document list, review & create
Airparser (Admin) — schema management, transformation rules, upload
