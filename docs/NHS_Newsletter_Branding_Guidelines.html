Newsletter Branding Guidelines — v1.0

Introduction

01

This document defines the visual identity standards for all New Home Solutions email newsletters and external communications. Consistent use of typography, colour, spacing, and layout ensures our brand is instantly recognisable, professional, and trustworthy across every recipient's inbox.

All newsletters, transactional emails, and partner communications must follow these guidelines. Any deviation requires approval from the Marketing or Operations team.

All email communications must use Futura LT Light for the “New Home Solutions” logo only, and Aptos for all other content. No other fonts are permitted.

Do & Don’t

08
Do
  • Use Futura LT Light for the “New Home Solutions” logo
  • Use Aptos for all body, headings, and content text
  • Use inline styles on every HTML element
  • Use table-based layouts for email compatibility
  • Use the sage/navy colour palette consistently
  • Include the header bar, accent stripe, and footer
  • Include the disclaimer and registered office text
  • Use the “Save the Trees” message in the footer
  • Maintain 640px email width
  • Use alternating row colours in data tables
  • Test in Outlook, Gmail, and Apple Mail
Don’t
  • Use Futura LT Light for anything other than the logo
  • Use Arial, Times New Roman, or any other unapproved font
  • Use CSS variables, classes, or <style> blocks
  • Use div-based layouts (use <table> instead)
  • Use colours outside the approved palette
  • Omit the NHS logo header or footer
  • Use bright or neon colours for emphasis
  • Use decorative fonts or custom web fonts
  • Exceed 640px width or use fluid widths
  • Use background images (not reliable in Outlook)
  • Send without including the legal disclaimer
